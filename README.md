## What I'm building

### ⛓ [BTC Predictor](https://btcpredictor.io) — AI Bitcoin Trading Bot
An autonomous AI agent that predicts Bitcoin direction and executes trades on Kraken Futures.
Every signal committed on-chain to Polygon PoS before execution — publicly auditable, forever.

- **614+ signals** generated and verified on-chain
- **Dual-gate system**: LLM (Claude) + XGBoost must agree before any trade
- **13 data sources**: on-chain metrics, macro calendar, technical analysis, sentiment
- Fully open source — `btc_predictions` repo is public

> *"If an AI trades, the blockchain should know about it before the exchange does."*

### 🏛 [AUREO](https://btcpredictor.io/aureo) — Italian Financial Education
Italy ranks **3rd-to-last in Europe** for financial literacy (OCSE 2023).

AUREO exists to change that — free, forever, on-chain verified.

- AI-generated content reviewed by an economist
- Every statistic cited with official source (ISTAT, Banca d'Italia, Consob)
- All claims hashed on Polygon before publication
